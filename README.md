# @amlplugins/twilio-conversations

Native `twilio` SDK re-exported under the `@amlplugins` namespace.

> Twilio Conversations — cross-channel chat (SMS, MMS, WhatsApp, chat) with participants and webhooks.

## Install

```bash
npm install @amlplugins/twilio-conversations
```

## Use

```ts
import { /* named exports from twilio */ } from "@amlplugins/twilio-conversations";
// or
import sdk from "@amlplugins/twilio-conversations";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
